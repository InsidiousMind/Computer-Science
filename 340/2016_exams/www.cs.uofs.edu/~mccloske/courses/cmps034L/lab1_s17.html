Activity #2: Submit background information

Take five minutes to compose a brief description of your experience using computing devices. In particular, if you've ever done any computer programming, indicate how much, in what context (e.g., a course in high school), and in which programming language(s).

CMPS 134 is an introductory course, so don't be concerned if you barely even know what "programming" means. Typically, more than half the students enrolled in the course have never programmed before.

Use Wordpad, MS Word, or some similar document editing tool to prepare your description. (Or you can wait until installing jGrasp in the next activity and then use its editing capabilities to do it.) It need only be a paragraph or two in length, but it is expected to be well written (i.e., using proper grammar and coherent sentences), such as would be expected of any capable college student.

When you have finished, submit your file by clicking here and following these instructions:

The web page that pops up should invite you to log in. Use the "Select" drop down menu to find your name, choose it, then enter your password where appropriate and click the "SUBMIT FORM" button. The page that should appear says Submit Page at the top. (This awkward phrase is meant to indicate that this is a web page to be used for submitting work to your instructor.)

You can enter a "Title" and/or a "Message" in the text boxes labeled as such, but that is not necessary. What is important is that you click on any of the "Browse" buttons underneath the "Message" box, select the file containing the document you just prepared, and then click on the "SUBMIT FORM" button.

You should get a "message submitted!!!" message. Use the web browser's BACK button to go back one page, and then refresh that page (by clicking on the refresh icon) (and telling the browser to "resend" if it asks). There should be a hyperlink on the page to what is now your instructor's copy of the file that you just submitted. You can click on it, which will cause the file to be downloaded onto your computer, if you want to verify that it is, indeed, a copy of the file that you submitted.

Activity #3: Installing Java JDK and jGrasp

  1. To install the Java Development Kit (JDK):
    • click here. On the page that pops up, click the big button (near top left of page) with the word "Java" in it with the image of a cup of coffee to its left.
    • On the page that pops up, look in the section headed by the phrase Java SE Development Kit 8u121 to find the Product/File Description (left column) appropriate for your computer. If you have an Apple computer, that would most likely be Mac OS X. If you have a computer that runs under Windows, it would probably be Windows x64 (but if you have an old "32-bit" machine, it would be Windows x86). Click on the corresponding link in the Download column in order to download the installation program.
    • Now run the installation program (e.g., by double clicking on the icon representing that program). During the installation process, you'll need to accept the license agreement and make some choices as to settings. For the latter, you are encouraged to choose the defaults.

      Take notice of the step that shows precisely where (i.e., in which folder/subdirectory) in your computer's file system Java is being installed. On a Windows machine, most likely it will be in C:\Program Files\Java\jdk?????, where the ?'s refer to the exact version number of JDK that you are installing. When installation has finished, this folder will have a folder within it named "bin" that contains (executable) files named javac.exe and java.exe, which are the Java compiler and the JRE (Java Runtime Environment?), respectively. If you can find these files in this location, you can be reasonably sure that you've completed this step correctly.

  2. To install jGrasp:
    Click here. Near the upper right corner of the page that appears will be a link labeled Download jGRASP. Click on it, and on the resulting page scroll down a little to find the section indicating jGrasp 2.0.3_05. Depending upon your computer's operating system, click on the appropriate green button to download an installation program. Then run it! (If your computer runs under Linux, you'll first have to "unzip" the compressed installation program file.)

    Again you may need to accept the license agreement and you are encouraged to accept the default settings as you proceed.

  3. Verify that the previous steps were successful by downloading, compiling, and executing the HelloWorld Java application program. Before doing that, you may want to create a folder/subdirectory on your computer's file system specifically for storing files relevant to this course.

    Having downloaded the HelloWorld.java file, click on its icon. Depending upon your computer's settings, the file may be loaded into jGrasp automatically. Or it might ask you to choose an application in which to load the file, in which case you should choose jGrasp.

    Once you've got the HelloWorld program loaded into jGrasp, compile and run it. (To compile, click on the button that looks like a green cross. To run, click on the button to its right, which looks like a humanoid stick figure in the act of running.)

    After you've run the program successfully, insert a new statement immediately after (or before) the statement that prints Hello, World and have the new statement print some other message. Compile and run the updated program. Show the TA that you have done this!
