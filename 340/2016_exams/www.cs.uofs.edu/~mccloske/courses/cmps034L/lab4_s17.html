Activity #1: Practice-It Self-Checks and Exercises on for-loops

After logging in to Practice-It, click on the link labeled Building Java Programs, 4th Edition and then on the link indicating BJP Chapter 2.

Do the following Self-Checks and, when finished, show your GTA (graduate teaching assistant) that you've completed all of them correctly. (Go back to the web page that lists all the Chapter 2 Self-Checks and refresh it. Any that were answered correctly will be indicated on the page.)

  1. BJP4 Self-Check 2.26 (fingerTrap): Show output produced by the execution of a given Java code segment involving a for-loop. Recall that \\ is an "escape sequence" that evaluates to the character \.

  2. BJP4 Self-Check 2.27 (howManyLines): Show output produced by the execution of a given Java for-loop. The notation i++ is a shorthand way of saying i = i + 1.

  3. BJP4 Self-Check 2.28 (blastOff): Show output produced by the execution of a given Java code segment involving a for-loop. The notation i-- is a shorthand way of saying i = i - 1.

  4. BJP4 Self-Check 2.29 (multiplication): Show output produced by the execution of a given Java for-loop (that has a loop nested inside it). The notation j++ is a shorthand way of saying j = j + 1. Don't forget the difference between the behaviors of System.out.print() and System.out.println()

  5. BJP4 Self-Check 2.30 (nestedLoops): (optional) Show output produced by the execution of a given Java program. The figure that it "draws" should be familiar to you from our recent lecture. The ++ operator is as mentioned above.

  6. BJP4 Self-Check 2.24 (oddStuff): Show output produced by the execution of a given Java code segment involving a for-loop.
    What is tricky here is that the upper bound of the range of values through which the loop control variable (count) iterates is defined by a variable (number) whose value changes from one loop iteration to the next. For this reason, I would argue that this is an inappropriate use of a for-loop.

  7. BJP4 Self-Check 2.31 (starExclamation1): Show output produced by the execution of a given Java code segment involving for-loops nested three levels deep.

  8. BJP4 Self-Check 2.32 (starExclamation2): Same as previous problem, except that the statement that prints an exclamation point is no longer in the body of the loop nested one level deep.
  9. BJP4 Self-Check 2.33 (starExclamation3): Same as previous problem, except that the statement that prints an exclamation point is now in the body of the most deeply-nested loop.
  10. BJP4 Exercise 2.6 (numberTriangle): Develop a pair of for-loops, one nested in the other, that "draws" the given right triangle. Not surprisingly, code for doing this will closely resemble that in the drawRightTriangle() method in any version of the DrawFiguresWithLoops application (to which there are links on the course web page).
