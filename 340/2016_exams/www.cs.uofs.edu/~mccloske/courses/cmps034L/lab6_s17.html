  5. BJP4 Self-Check 4.20 (charMystery): Recall that the statement letter++ increases the value of variable letter by one. Keep in mind that values of type char are pseudo-integers in the sense that each of them is associated to an integer value that represents it. Thus, for example, if letter has value 'D', then execution of letter++ changes its value to 'E' (because the number that represents 'E' is one greater than the number that represents 'D').

(Assume that February always has 28 days.)
