Programming Assignment #1: Freeing Java Arrays from the [0,N) Index Range Straitjacket
Due: Noon, Monday August 28

Requirements

For this assignment, you are to complete the development of the Java class DynAnchArray (an abbreviation for "Dynamically Anchored Array"), whose instances have essentially the same functionality as (one-dimensional) arrays (even if the syntax for accessing their elements more closely resembles that of ArrayLists), except that their index ranges are specified, at construction time, by the client. (In Java, every array's index range begins at zero, which can be an inconvenience.)

Each method that needs to be completed is identified by the comment // STUB!!.

For your convenience, the Java application DynAnchArrayTester is provided to you. In order to make fruitful use of it, you must pass to it a sequence of command line arguments (or what jGrasp calls "run arguments"). How to do this when running a program in jGrasp is explained here.

What DynAnchArrayTester does is to place whatever command line arguments are provided to it into an instance of the DynAnchArray class, in a pseudo-random order. (The bottom of the index range is also chosen pseudo-randomly.) Then it displays the contents of that object two times, once using its toString() method and once using an Iterator. 

% java DynAnchArrayTester Kirk Spock Uhura McCoy Scott Sulu
Contents of a DynAnchArray as provided by its toString() method:

-2: Spock
-1: Uhura
0: Sulu
1: Kirk
2: McCoy
3: Scott

Contents of DynAnchArray as provided by an Iterator:

Spock
Uhura
Sulu
Kirk
McCoy
Scott

Goodbye.

To illustrate, on the right is shown an invocation of DynAnchArrayTester from the command line in a UNIX environment (shown in boldface) and the output that results.

In case you are not familiar with some of the features of Java that are to be used in this assignment (e.g., generics, nested classes, iterators), provided to you is this sample Java class: StackViaAryBounded.
