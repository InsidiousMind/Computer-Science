Programming Assignment #1: Unbounded Natural Numbers

For this assignment, you are to complete the Java class NaturalNumber1, which is a subclass of the abstract class NaturalNumber. Notice that some of the methods in the latter are complete. Incomplete public methods in the former are indicated by the comment STUB!!.

You may wish to add private methods to NaturalNumber1 and you will certainly have to add declarations of instance variables there.

Perhaps the most "natural" way to represent a natural number is using an array digits[] of byte values, where each element in the array stores one digit of the number. For example, an object representing the natural number 8403426 could have a state that "looks" like this: 

         0   1   2   3   4   5   6
       +---+---+---+---+---+---+---+
digits | 6 | 2 | 4 | 3 | 0 | 4 | 8 |
       +---+---+---+---+---+---+---+

The digit values are stored in the array from least to most significant because it simplifies the implementation of the relevant algorithms. (Using elements of type byte was suggested because each one occupies only a single byte of memory rather than four, as do values of type int.)

Now, this example assumes a representation based upon the decimal (i.e., base 10) numeral system. More efficient would be to use base 256, as each element of the array (assuming elements of type byte) can store any value in the range 0..255 and thus a large number N could be stored using only ⌊log256 N⌋ + 1 elements rather than ⌊log10 N⌋ + 1 elements, the latter of which is a little more than twice as many.
