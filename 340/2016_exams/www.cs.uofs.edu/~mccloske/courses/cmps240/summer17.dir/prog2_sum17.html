Programming Assignment #2: Freeing Java Arrays from the Index Range Straitjacket

For this assignment, you are to develop a Java class whose instances have the same functionality as arrays, except that their index ranges are specified, at construction time, by the client. (In Java, every array's index range begins at zero.)

For the purposes of this description, this class shall be called Array0. (Suggestions for a better name are welcomed!) Here is a skeleton: 

   public class Array0 {


      //*************  C o n s t r u c t o r  *************

      /** Establishes the lower bound of the array's index range and its
      **  length.
      **  pre: length >= 0
      */
      public Array0(int indexLow, int length) { ... }

      
      //*************  O b s e r v e r s  *************


      /** Returns the lower bound of the array's index range.
      */
      public int indexLowerBound() { ... }

      /** Returns the upper bound of the array's index range.
      */
      public int indexUpperBound() { ... }

      /** Returns the array's length.
      */
      public int length() { ... }

      /** Returns the item at the specified position of the array.
      **  pre: indexLowerBound() <= k <= indexUpperBound()
      */
      public Object get(int k) { ... }


      //*************  M u t a t o r  *************

      /** Places the specified item at the specified position of the array.
      **  pre: indexLowerBound() <= k <= indexUpperBound()
      */
      public void put(int k, Object obj) { ... }

   }

An IndexOutOfBoundsException (see the package java.lang) should be thrown in case a pre-condition is not met.

As an improvement, the class should be made to be generic so that the client can choose what type of values are to occupy any particular array that it (i.e., the client) creates. Modifying the above skeleton to "generify" it, we get 

   public class Array0<T> {

      //*************  C o n s t r u c t o r  *************

      /** Establishes the lower bound of the array's index range and its
      **  length.
      **  pre: length >= 0
      */
      public Array0(int indexLow, int length) { ... }

      
      //*************  O b s e r v e r s  *************


      /** Returns the lower bound of the array's index range.
      */
      public int indexLowerBound() { ... }

      /** Returns the upper bound of the array's index range.
      */
      public int indexUpperBound() { ... }

      /** Returns the array's length.
      */
      public int length() { ... }

      /** Returns the item at the specified position of the array.
      **  pre: indexLowerBound() <= k <= indexUpperBound()
      */
      public T get(int k) { ... }


      //*************  M u t a t o r  *************

      /** Places the specified item at the specified position of the array.
      **  pre: indexLowerBound() <= k <= indexUpperBound()
      */
      public void put(int k, T obj) { ... }

   }
