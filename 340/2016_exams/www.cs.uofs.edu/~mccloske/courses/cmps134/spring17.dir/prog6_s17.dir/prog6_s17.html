Background

For this assignment, you are to complete the development of the Java class MultipleChoiceTestScorer. An object/instance of this class is for the purpose of scoring a multiple choice test taken by a group of students.

In creating a MultipleChoiceTestScorer object, the client must provide to the constructor, via a String parameter, the answer key of the test.

Observer methods in the class answer such questions as

  • How many students' tests have been scored so far?
  • What raw score (number of questions answered correctly) was achieved by a particular student?
  • What is the average of the students' raw scores?
  • What is the standard deviation of the students' raw scores?
  • How many students answered a particular question correctly?

The lone mutator method is scoreTest(), and it is the key to making objects of this class work correctly. This method is responsible for scoring a test (the answers to which are provided via a parameter) and for updating the instance variables appropriately. (Note that all four instance variables are subject to change each time scoreTest() is called.)

An Example

To illustrate, suppose that the test being scored has as its answer key "bacad" and that the test papers of students named "Emily", "Jim", and "Diane" have been scored so far, on which their answers were "bbcad", "adcbd", and "bacad", respectively. As the instance variables indicate, three tests have been scored so far, submitted by students named Emily, Jim, and Diane, who achieved raw scores 4, 2, and 5, respectively. Also, the five questions on the test were answered correctly twice, once, thrice, twice, and thrice, respectively. Because only three tests have been scored, the values in array segment rawScores[0..2] are of significance, but those in the rest of the array are irrelevant, as indicated by the dashes. The same applies to the stuIDs[] array.

Notice that elements of stuIDs[] and rawScores[] in corresponding locations are related to each other. Specifically, for each relevant value of k, stuIDs[k] and rawScores[k] contain values pertaining to the k-th test to be scored, namely the ID of the student who took it and the raw score earned by that student, respectively. For this reason, these arrays are informally said to be "parallel". On the other hand, the correctCount[] array is parallel to neither of the others, as the value stored in its location k pertains not to the k-th test to be scored but rather to question k on the test.

Average and Standard Deviation

Two of the observers in the class are to return, respectively, the average (more precisely, the arithmetic mean) and the standard deviation of the raw scores of the already-scored tests.

With respect to the example shown in the previous section (in which three tests have been scored, with raw scores 4, 2, and 5), the values returned by the averageRawScore() and stdDevOfRawScores() methods should be something close to 3.6667 and 1.247, respectively.
