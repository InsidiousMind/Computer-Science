Programming Assignment #1: There Was an Old Lady Who Swallowed a Fly
Due Time: 11:59pm, Thursday, Feb. 16

Requirements

You may be familiar with the nursery rhyme There was an Old Lady Who Swallowed a Fly, the lyrics of which, like those of the song The Twelve Days of Christmas, are said to be "cumulative".

For this assignment, you are to develop a Java application that produces as output the lyrics of a somewhat modified version of the rhyme, exactly as they are shown at the bottom of this document. A skeleton is provided for you to get a start.

Your solution should show evidence that you generously employed the technique of procedural decomposition, which should help you to obtain a program that is well-structured and in which there is very little code redundancy. It is suggested that, for each verse, you introduce a method that prints it. Try to avoid having two print (or println) statements that print the same string.

Submission of Program

Submit your source code file (which must be named LadySwallowedFly.java) by clicking on the Submit/Review link (on the course web page) that is adjacent to the link that brought you to this page. Do not submit the associated .class file.

The skeleton that was provided to you (see link above) includes a comment describing the purpose/behavior of the program; a comment identifying the course, semester, and assignment number; and comments intended to identify the program's author (you!) and those who helped you to develop the program. Also included is a comment intended for you to describe any known deficiencies of your program. Consider this to be a template that you are expected to follow in all subsequent assignments.
