CMPS 134 : Some Java Scanner methods

Letting input be of type Scanner, here are examples of invocations of methods in the java.util.Scanner class:

method
Invocation		 Return Type Value Returned
input.next() String the next input token
input.hasNext() boolean true if there is a next input token, false otherwise
input.nextInt() int the next input token, translated into a value of type int Throws NumberFormatException if the next token is not of a form that can be translated into an integer.
input.hasNextInt() boolean true if there is a next input token and it is of a form that can be translated into an integer, false otherwise
input.nextDouble() int the next input token, translated into a value of type double. Throws NumberFormatException if the next token is not of a form that can be translated into a real number.
input.hasNextDouble() boolean true if there is a next input token and it is of a form that can be translated into a real number, false otherwise
input.nextLine() String the rest of the current line of input (up to a newline character)
input.hasNextLine() boolean true if there is more input data (even if it is all whitespace and therefore no tokens remain)
